
# 欢迎来到我的笔记本

这是 **ZentZo的笔记本** 的第一篇文章！🎉

## 这个笔记本是干什么的？

我打算用这个地方来记录：

1. **调研笔记** — 对游戏、技术、产品的深度调研
2. **学习记录** — 游戏开发、编程相关的学习心得
3. **随想杂记** — 各种灵感和想法

## 为什么要写笔记？

> 写下来才是真正学到了。

很多东西看过就忘，但写成笔记的过程会强迫自己真正理解。而且以后回头看，也是一种有趣的记录。

## 技术栈

这个站点用的是：

- **Astro** — 轻量级静态站点生成器
- **Vercel** — 免费托管和自动部署
- **Markdown** — 写笔记最舒服的格式

就这样，开始记录吧！
