# 《枪火重生》（Gunfire Rebirth）Build 机制深度调研报告

## 执行摘要
《枪火重生》的 Build 构筑是一个典型的“多维相乘”系统。玩家通过**英雄觉醒（金爵）**提供框架，**秘卷**提供变量，**武器铭刻**进行最终输出输出。在进入高难度（轮回模式）后，**灵佑**与**灵气连接**成为了 Build 的质变引擎。本报告将深度拆解这些系统的工作原理及其相互作用。

---

## 1. 构筑核心架构：三大支柱
一个成功的 Build 由以下三个系统共同决定：

### 1.1 英雄觉醒（Ascensions）- “构筑的骨架”
英雄觉醒（金爵）是每局游戏的起点，决定了你这一局的战斗风格。
- **设计逻辑**：每位英雄有三条路径（例如：主技能强化、次要技能强化、武器/生存强化）。
- **提升曲线**：觉醒分为 1-3 级。1-2 级通常是数值提升，3 级则引入**机制质变**（如青燕跃击击杀重置 CD）。
- **联动效应**：觉醒会彻底改变技能逻辑。例如雷落的“落雷之怒”将射击与落雷频率挂钩，实现了武器与技能的深度耦合。

### 1.2 秘卷（Occult Scrolls）- “构筑的变量”
秘卷提供全局被动，通过稀有度（普通、精良、传说、诅咒）区分强度。
- **核心逻辑**：秘卷的协同（Synergy）是构筑成败的关键。例如“血肉之躯”与“火焰狂热”的吸血组合。
- **构筑流派**：
    - **防御流**：异能之躯（全盾流） vs 血肉之躯（全血流）。
    - **机制流**：如“高级弹库”取消换弹，直接改变武器的使用效率。

### 1.3 武器铭刻（Inscriptions）- “构筑的执行端”
武器是伤害的最终载体，铭刻决定了武器的上限。
- **铭刻分级**：绿色（普通）、蓝色（稀有）、橙色（专属）、紫色（双子）。
- **双子铭刻（Gemini）**：Build 成型的标志。需要主副武器携带相同双子词条。
    - **共享元素**：让一把枪同时打出两种元素，触发融合反应。
    - **共享弹夹/暴击**：极大提升武器基础性能。

## 2. 核心底层机制：伤害与元素

### 2.1 伤害计算逻辑
《枪火重生》的伤害提升遵循以下公式（简化版）：
> **总伤害 = 基础伤害 × (1 + 加法加成总和) × (乘法加成 1) × (乘法加成 2)...**

- **加法加成**：如大部分秘卷提供的“武器伤害 +50%”。
- **乘法加成（最终伤害）**：如某些觉醒或词条注明的“最终伤害增加”。
- **幸运一击**：不同于暴击，是概率触发的伤害翻倍（黄字 -> 橙字 -> 紫字）。

### 2.2 元素交互系统（Elemental Fusion）
这是 Build 构筑中放大伤害的最快手段：
| 元素组合 | 反应名称 | 效果 |
| :--- | :--- | :--- |
| **火 + 毒** | **爆炸 (Explosion)** | 造成高额范围伤害。 |
| **火 + 雷** | **傀儡 (Manipulation)** | 敌人反戈相向，攻击友军。 |
| **雷 + 毒** | **瘴气 (Miasma)** | 极高层数的百分比掉血（清场神技）。 |

## 3. 高阶构筑：轮回模式特有系统
在轮回模式（Reincarnation）中，构筑的重点转向了精魄消费。

### 3.1 灵佑（Spiritual Blessings）
开局三选一的强力天赋，通常是一个 Build 的**核心（Core）**。
- 例如：太子的“元素支配”让烟雾弹变为全元素伤害，直接奠定全屏融合流。

### 3.2 灵气连接（Spiritual Links）
当玩家持有特定组合的秘卷、觉醒或武器时，会触发额外的“羁绊”效果。这为后期溢出的资源提供了新的转化维度。

---

## 4. 经典 Build 案例拆解

### 4.1 孢子流（Weapons: Spore Build）
- **核心**：双子铭刻“孢子共享”。
- **原理**：一把高攻速武器（如彩虹）触发，一把高单发伤害武器（如锐弩）叠层。孢子伤害基于两把武器的属性，无视距离。
- **适用**：所有英雄，对觉醒要求极低。

### 4.2 元素无限流（Hero: Li/Crown Prince）
- **核心**：灵佑“元素支配” + 秘卷“元素织网” + 双子“元素共享”。
- **原理**：通过副手武器覆盖缺失元素，实现常驻三种融合反应，利用爆炸或瘴气瞬间清场。

---

## 5. 调研结论与建议
- **构筑逻辑**：前期拿生存秘卷，中期找核心觉醒，后期强凑双子铭刻。
- **工匠利用**：武器 5 级后务必进行“蚀刻”以获取双子词条，这是进入第三章后的首要任务。
- **奇货商（Peculiar Chest）**：利用奇货商进行秘卷重铸，将废卷转化为核心卷轴。
