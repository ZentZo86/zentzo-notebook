# 《Roboquest》（酷炫机器人）Build 机制深度调研报告

## 执行摘要
《Roboquest》的 Build 构筑强调“高速机动”与“动作协同”。玩家通过选择**英雄天赋（Perks）**构建战斗风格，利用**武器词条（Affixes）**和**全局道具（Items）**强化数值。与《枪火重生》相比，本作的构筑更依赖于操作触发（如踩头、近战击杀）和瞬间的爆发反馈。

---

## 1. 构筑核心：英雄天赋（Perks）与升级系统
天赋系统是 Build 的灵魂，决定了角色的主动技能如何演化。

### 1.1 天赋等级节点
玩家在关卡中通过收集“虹膜碎片”升级。关键节点如下：
- **核心天赋（Perks）**：在 **2、4、7、10** 级时提供三选一的职业专属或通用天赋。每轮游戏通常仅能持有 4 个核心天赋 [1]。
- **附属升级（Extensions）**：在核心天赋选定后，后续等级（如 3、5、6 等）会产出针对该核心天赋的二选一细分强化。
- **质变效应**：核心天赋能彻底改变职业定位。例如，工程师的“废料机枪”能将辅助无人机变为手持高火力武器，从召唤流转为射击流 [1][2]。

---

## 2. 装备系统：武器词条与全局道具

### 2.1 武器系统（Weapons）
- **稀有度分级**：普通、罕见、卓越、史诗、惊艳。高稀有度武器固定带有副攻击（Alt-Fire）模式 [1]。
- **词条（Affixes）**：
    - **固定词条**：如“重型”（移速降，伤害升）、“双持”。
    - **随机词条**：如“自动追踪”、“幸运”（20% 自动爆击）。
- **暴击机制**：精准类武器（Sniper/Bow）拥有极高的爆击倍率（最高 3.75x），强调弱点打击 [1][2]。

### 2.2 全局道具（Items）
- **获取方式**：在商店消耗能量电池购买，整场常驻。
- **腐化道具（Corrupted Items）**：提供极高增益但伴随负面惩罚（如增加伤害但降低 15% 移速），需要玩家通过天赋中的机动性加成来对冲惩罚 [3]。

## 3. 底层机制：元素反应与协同

### 3.1 元素系统（Elemental System）
| 元素 | 效果 | 进阶效果（Mastery） |
| :--- | :--- | :--- |
| **燃烧 (Burn)** | 持续伤害 (DoT) | 概率向 6m 内敌人扩散。 |
| **电击 (Shock)** | 造成僵直 (Stun) | 击晕时产生额外范围伤害。 |
| **冰冻 (Cryo)** | 冻结目标 (Freeze) | 对冻结敌人造成额外爆击伤害。 |

### 3.2 协同逻辑（Synergy）
- **元素协同**：道具“一块蛋糕”能对冻结敌人施加燃烧，完美配合元素师（Elementalist）的多元素覆盖流。
- **动作协同**：道具“绿袜子”允许通过踩头恢复划痕血量，与游骑兵（Ranger）的高机动构筑高度契合 [3][4]。

## 4. 2024-2025 赛季经典 Meta 构筑

### 4.1 魔法师流 (Elementalist: Wizard Build)
- **核心**：放弃主武器，全堆技能冷却和元素灌注（Infusion）。
- **玩法**：利用元素连击快速刷新主动技能，实现全屏法术轰炸。

### 4.2 自动爆击流 (Superbot/Ranger: Full Crit)
- **核心**：寻找带有“幸运”蓝色词条的武器 + 道具“鹰眼瞄具”。
- **玩法**：配合游骑兵的“暗影突袭”在隐身状态下打出惊人的首发爆击伤害。

### 4.3 工程师召唤流 (Engineer: Drone Army)
- **核心**：天赋“狂暴无人机” + 道具“备用零件”。
- **玩法**：最大化无人机数量与持续时间，玩家负责走位避弹，由召唤物完成清场。

---

## 5. 调研结论与操作建议
- **节奏把控**：由于 10 级后核心天赋不再产出，前中期的天赋选择决定了结局。
- **电池优先级**：优先购买核心道具（Items），其次才是升级武器等级。低品质但好词条的武器值得花钱升到后期。
- **平衡惩罚**：在捡取腐化道具前，务必确认当前 Build 是否有足够的 Perk 点数抵消其副作用。
